🗄️ Crear la base de datos en Supabase (gratis)

1
Crear cuenta gratuita Ve a supabase.com → Start your project → Sign up con Google o email
2
Crear un nuevo proyecto Clic en "New Project" → Elige un nombre (ej: "perfumeria-bot") → Elige región → Create new project
3
Ir al SQL Editor En el menú izquierdo busca: SQL Editor (ícono de código)
Panel Supabase — menú izquierdo:
🏠 Home
📊 Table Editor
📝 SQL Editor ← AQUÍ
🔐 Authentication
⚙️ Settings
4
Pegar y ejecutar el script SQL Abre el archivo supabase-setup.sql de tu proyecto, copia TODO el contenido y pégalo en el editor → clic en "Run" (botón verde)
-- Esto crea las 4 tablas automáticamente: ✅ ventas (registro de ventas del bot) ✅ productos (catálogo de perfumes) ✅ config (configuración del bot) ✅ ventas_pos (datos de VectorPOS)
5
Obtener las credenciales Ve a Settings → API en el menú izquierdo
Settings → API — copia estos dos valores:
📌 Project URL
https://xxxxxxxxxxx.supabase.co
↳ Esto es tu SUPABASE_URL

🔑 anon / public key
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
↳ Esto es tu SUPABASE_KEY
6
Pegar en el archivo .env Copia esos dos valores y pégalos en el .env como se muestra en la pestaña de Instalación
✅ ¡Supabase listo! La base de datos guarda las ventas automáticamente en la nube.
